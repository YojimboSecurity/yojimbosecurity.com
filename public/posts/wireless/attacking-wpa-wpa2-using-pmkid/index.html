Attacking WPA/WPA2 Using PMKID

By David Johnson

April 23, 2019

Having heard about the new attack on WPA/WPA2 using PMKID I am attempting to reproduce.

I am using a raspberrypy with an Alfa wireless card. The AP is a tplink N600. Also installed on system76 Ubuntu18 The first thing I did was clone the repos.

Tools

Hcxdumptool

git clone https://github.com/ZerBea/hcxdumptool.git

cd hcxdumptool/

make

sudo make install

Hcxtools

git clone https://github.com/ZerBea/hcxtools.git

cd hcxtools/

sudo apt-get install libcurl4-openssl-dev libssl-dev zlib1g-dev libpcap-dev

make

sudo make install

Hashcat

git clone https://github.com/hashcat/hashcat.git

or

note: did not seam to work in pi and for pi it was hashcat-data

sudo apt install hashcat
